# 🧠 Generador de Tablas de Verdad

Aplicación web desarrollada con Streamlit que permite generar tablas de verdad a partir de expresiones lógicas.

## 🚀 Características

- Ingreso de expresiones lógicas (AND, OR, NOT, XOR)
- Generación automática de variables
- Tabla de verdad completa (2ⁿ combinaciones)
- Resultados en formato 0 y 1
- Descarga de la tabla en formato CSV
- Manejo de errores en expresiones inválidas

## 🛠️ Tecnologías usadas

- Python
- Streamlit
- Pandas
